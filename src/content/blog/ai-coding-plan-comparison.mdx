## 🔍 详细对比

### 1. GitHub Copilot

GitHub Copilot 是目前最流行的 AI 编程助手之一，深度集成 GitHub 生态系统。

#### 套餐详情

| 套餐 | 月费 | 年费 | 适合人群 |
|------|------|------|----------|
| **Free** | $0 | - | 初学者、轻度用户 |
| **Pro** ⭐ | $10 | $100 | 个人开发者（最受欢迎） |
| **Pro+** | $39 | $390 | 重度用户、需要更多模型 |

#### 使用限制

- **Free 版**：基础功能，有限使用
- **Pro 版**：无限代码补全，支持多模型
- **Pro+ 版**：所有 Pro 功能 + 代理模式 + 更多模型选择

#### 支持的 IDE

- VSCode
- Visual Studio
- JetBrains IDEs（IntelliJ、PyCharm 等）
- Vim/Neovim
- **OpenClaw** ✅

> 📌 **数据来源**: [GitHub Copilot Plans](https://github.com/features/copilot/plans)

### 2. Cursor

Cursor 是一款基于 VSCode 的 AI 代码编辑器，支持多种 AI 模型切换。

#### 套餐详情

| 套餐 | 月费 | 适合人群 |
|------|------|----------|
| **Hobby** | $0 | 初学者、轻度用户 |
| **Pro** | $20 | 个人开发者 |
| **Pro+** | $60 | 重度用户 |
| **Ultra** | $200 | 团队/企业 |

#### 使用限制

| 套餐 | Agent 请求 | Tab 补全 | 模型使用量 |
|------|------------|----------|------------|
| Hobby | 有限 | 有限 | 基础 |
| Pro | 扩展 | 无限 | 标准 |
| Pro+ | 扩展 | 无限 | **3倍**（OpenAI、Claude、Gemini）|
| Ultra | 扩展 | 无限 | **20倍**（OpenAI、Claude、Gemini）|

#### 核心功能

- 🔄 多模型切换（OpenAI、Claude、Gemini）
- ☁️ 云端 Agent
- 📐 最大上下文窗口
- 🎯 无限 Tab 补全（Pro 及以上）

#### 支持的 IDE

- Cursor 本身（基于 VSCode）
- **OpenClaw** ✅（通过 API）

> 📌 **数据来源**: [Cursor Pricing](https://cursor.com/pricing)

## 🤔 如何选择？

### 决策树

1. **你主要使用什么 IDE？**
   - VSCode → Cursor 或 GitHub Copilot
   - JetBrains → GitHub Copilot 或通义灵码
   - OpenClaw → 全部支持 ✅

2. **你的主要编程语言是？**
   - 多种语言 → GitHub Copilot（支持最广）
   - 中文场景多 → 通义灵码
   - 需要切换模型 → Cursor

3. **你的使用强度？**
   - 偶尔用 → 免费版足够
   - 每天用 → Pro 版性价比最高
   - 重度依赖 → Pro+ 或更高

4. **你的预算？**
   - 0元 → 免费版组合
   - ¥50-100 → 单平台 Pro
   - ¥200+ → 多平台或高级套餐
