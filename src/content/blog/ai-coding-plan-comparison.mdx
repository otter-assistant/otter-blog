# AI Coding Plan 对比指南 🤖

> **什么是 Coding Plan？** Coding Plan 是 AI 模型厂商推出的编程订阅套餐，让你可以在 OpenClaw、Cline、Cursor 等 AI 编程工具中使用他们的模型。本文对比支持 OpenClaw 的主流 Coding Plan。

## 🎯 为什么需要 Coding Plan？

如果你使用 **OpenClaw**、Cline、Claude Code 等 AI 编程工具，你需要：
1. 订阅一个 **Coding Plan** 获得模型调用额度
2. 在工具中配置 API Key
3. 即可开始使用 AI 辅助编程

相比直接调用 API，Coding Plan 通常有：
- ✅ 更高的额度
- ✅ 更优惠的价格
- ✅ 更稳定的服务
- ✅ 无需担心网络限制

## 🔍 详细对比

### 1. 智谱 GLM Coding Plan ⭐ 推荐

智谱 GLM Coding Plan 是国内最成熟的编程订阅套餐，完美支持 OpenClaw。

#### 套餐详情

| 套餐 | 每5小时限额 | 每周限额 | 适合人群 |
|------|-----------|---------|----------|
| **Lite** | ~80 次 prompts | ~400 次 prompts | 轻度用户、学生 |
| **Pro** ⭐ | ~400 次 prompts | ~2000 次 prompts | 个人开发者 |
| **Max** | ~1600 次 prompts | ~8000 次 prompts | 重度用户、团队 |

> 💡 **说明**：1次 prompt 预计可调用模型 15-20 次

#### 核心优势

- 🚀 **速度快**：55+ tokens/秒，实时交互体验
- 🔧 **工具支持**：OpenClaw、Cline、OpenCode、Claude Code 等 10+ 工具
- 🌐 **无网络限制**：国内服务，稳定可靠
- 💰 **性价比高**：相当于月订阅费用的 15-30 倍 API 额度
- 🎁 **额外能力**：图像视频理解、联网搜索、网页读取、MCP 支持

#### 使用限制

- **5小时动态刷新**：额度消耗后 5 小时自动恢复
- **周限额**：自下单起 7 天为周期，额度刷新
- **仅限编程工具**：不能在非编程场景使用

#### 推荐理由

> 如果你使用 OpenClaw，智谱 GLM Coding Plan 是目前最成熟的选择。

## 📝 如何接入 OpenClaw？

### 智谱 GLM

1. 访问 [智谱 GLM Coding Plan](https://bigmodel.cn/glm-coding)
2. 选择套餐并订阅
3. 获取 API Key
4. 在 OpenClaw 中配置：
   - API Key: 你的智谱 API Key
   - Base URL: `https://open.bigmodel.cn/api/paas/v4/`
   - 模型: 选择 GLM-4 或 GLM-5

### 火山引擎豆包

1. 访问 [火山引擎控制台](https://console.volcengine.com/ark)
2. 开通 Coding Plan
3. 获取 API Key
4. 在 OpenClaw 中配置：
   - Base URL: 兼容 Anthropic/OpenAI 协议
   - 模型: `ark-code-latest`

## 🆚 Coding Plan vs AI IDE

很多人会混淆 **Coding Plan** 和 **AI IDE**，它们是不同的：

| 类型 | 例子 | 说明 |
|------|------|------|
| **Coding Plan** | 智谱 GLM、豆包 | 订阅套餐，用于在工具中调用模型 |
| **AI IDE** | Cursor、Windsurf | 自带 AI 功能的编辑器 |

**关键区别**：
- Coding Plan 是**额度订阅**，可以在多个工具中使用
- AI IDE 是**软件订阅**，只能在那个编辑器中使用

**最佳实践**：
- 如果你用 OpenClaw → 订阅 Coding Plan
- 如果你用 Cursor → 订阅 Cursor Pro
- 可以组合使用：Cursor Pro + 智谱 GLM Coding Plan

## ⚠️ 注意事项

### 智谱 GLM 使用规范

1. **仅限编程工具**：Coding 套餐只能在支持的编程工具中使用
2. **不可 API 独立调用**：不能以 API 形式单独调用（会单独计费）
3. **额度耗尽**：需要等待下一个 5 小时周期恢复，不会消耗账户余额
4. **账号规范**：一个账号只能订阅一个 Coding 套餐

### 火山引擎使用规范

1. 需要通过控制台开通服务
2. 具体限制请参考官方文档
