| 套餐 | 月费 | 每5小时限额 | 每周限额 | 适合人群 |

- 5小时动态刷新 + 每周限额
- 仅限编程工具使用

## ⚠️ 注意事项

### 通用规则

1. **仅限编程工具**：Coding Plan 只能在支持的编程工具中使用
2. **不可 API 独立调用**：单独调用 API 会单独计费
3. **账号规范**：一个账号通常只能订阅一个 Coding 套餐

### 各平台特点

- **智谱 GLM**：文档最完善，限制最清晰（5小时+周限额）
- **百度千帆**：新用户优惠最大（首月¥7.9）
- **MiniMax**：价格最低，调用量高
- **火山引擎**：支持多模型切换，但文档信息较少
